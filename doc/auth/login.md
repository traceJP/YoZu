# 账号登录
1. 需要提供唯一的 邮箱或手机号 + 密码

# 手机登录
1. 需要提供手机号 + 验证码
2. 当手机登录时若系统未存在相同的手机号，则会自动注册一个新用户


#  社交登录 前后端分离流程：
1. 用户点击qq登录按钮，跳转到qq授权页面
2. 用户未授权，通过qq配置回调到前端登录页面
3. 用户授权，通过qq配置同样回调到前端登录页面，但是会携带code参数
4. 前端判断code参数和带上对应的社交平台枚举，调用后端接口loginBySocial换取本系统token
